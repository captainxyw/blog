

# Spring Boot开发小而美的个人博客
**个人博客功能：**
* 分类管理
* 标签管理
* 博客管理
* 评论管理
* 博客归档


**技术组合：**

*  后端：Spring Boot + JPA + thymeleaf模板
*  数据库：MySQL
*  前端UI：Semantic UI框架


**工具与环境：**

*  IDEA
*  Maven 3
*  JDK 8


**课程内容模块：**

*  需求分析与功能规划
*  页面设计与开发
*  技术框架搭建
*  后端管理功能实现
*  前端管理功能实现


**插件集成:**

[编辑器 Markdown](https://pandao.github.io/editor.md/)

[内容排版 typo.css](https://github.com/sofish/typo.css)

[动画 animate.css](https://daneden.github.io/animate.css/)
